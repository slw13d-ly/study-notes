## 변수

- 변수 - 하나의 값을 저장할 수 있는 메모리 번지에 붙여진 이름
  - 변수를 통해 프로그램이 메모리 번지에 값을 저장 + 읽기 가능
  - 어딘가 값을 보관해두고 꺼낼 수 있는 저장소

### 변수 선언 방법

- 첫 글자 문자. 중간부터 문자, 숫자, $, \_ 등 가능
- 첫 문자는 소문자. 캐멀 스타일로 작성
  - 캐멀(camel) 스타일 - 낙타 등, 대소문자를 섞는 것
    - 클래스명 - Week.java, MamberGrade.java, ProductKind.java
    - 변수명 - score, mathScore, sportsCar

### 변수 선언 

[소스](src/variable/Var4.java)

- 저장되는 값의 타입과 이름만 결정, 메모리의 할당된 것 x
- 최초의 값이 대입될 시 메모리에 할당, 해당 메모리에 값 저장

### 변수 초기화

- 변수 선언 후, 선언한 변수에 처음으로 값을 저장

### 변수 값 저장 

[소스](src/variable/Var2.java)

- 대입 연산자( = ) 사용 - `우측 값을 좌측 변수에 대입`
- 변수 초기화 - 선언한 변수에 처음으로 값을 대입하는 것
  - 초기화되지 않은 변수는 메모리에 할당되지 않음 - 컴파일 에러
- 초기값 - 변수에 최초로 대입한 값
  - 변수 선언과 동시에 대입 가능

### 변수 값 읽기

- 변수의 이름을 적어주기만 하면 됨

### 변수 값 변경 

[소스](src/variable/Var3.java)

  ```java
  a = 10; //변수 a에 10 저장
  System.out.println(a); // 변수 a값 읽어옴, a => 10, 10 출력

  a = 50; // 변수 a값 50으로 변경, a(10 -> 50)
  System.out.println(a); // 변수 a값 읽어옴, a => 50, 50 출력
  ```
  - 변수 값 변경시 변수에 들어있던 기존 값은 삭제
