## 변수

- 하나의 값을 저장할 수 있는 메모리 번지에 붙여진 이름
- 변수를 통해 프로그램이 메모리 번지에 값을 저장 + 읽기 가능
- 어딘가 값을 보관해두고 꺼낼 수 있는 저장소

### 변수 명명 규칙

- 첫 글자 영문자(숫자 x). 중간부터 영문자, 숫자, $, \_ 등 가능
- 첫 문자는 소문자. 캐멀 표기법으로 작성
  - 캐멀(camel) 표기법 - 낙타 등모양, 공백을 쓸 수 없어 단어 구분을 위해 대소문자를 섞는 것
    - 클래스명 - Week.java, MamberGrade.java, ProductKind.java
    - 변수명 - score, mathScore, sportsCar
- 예약어 불가

### 변수 초기화

[소스](src/variable/Var5.java)

- 변수 선언 후, 선언한 변수에 처음으로 값을 저장
  1. 변수의 선언과 초기화를 각각 따로 가능
  2. 변수 선언과 동시에 초기화 가능
  3. 여러 변수 선언과 초기화 동시에 가능

- 초기화 하지 않는다면? - [소스](src/variable/Var6.java)
  - 컴파일 에러 발생 - 자바 문법이 맞지 않을 때 발생
  - 변수 선언 시 메모리를 차지 하는데, 선언되어 있지 않으면 기존에 어떤 값이 들어있었는지 모름
    → 초기화 하지 않으면 이상한 값이 출력될 수 있음. 예방을 위해 자바에서 변수 초기화를 강제함

### 변수 값 저장 

[소스](src/variable/Var2.java)

- 대입 연산자( = ) 사용 - `우측 값을 좌측 변수에 대입`
- 변수 초기화 - 선언한 변수에 처음으로 값을 대입하는 것
  - 초기화되지 않은 변수는 메모리에 할당되지 않음 - 컴파일 에러
- 초기값 - 변수에 최초로 대입한 값
  - 변수 선언과 동시에 대입 가능

### 변수 값 변경 

[소스](src/variable/Var3.java)

  ```
  a = 10; //변수 a에 10 저장
  System.out.println(a); // 변수 a값 읽어옴, a => 10, 10 출력

  a = 50; // 변수 a값 50으로 변경, a(10 -> 50)
  System.out.println(a); // 변수 a값 읽어옴, a => 50, 50 출력
  ```
  - 변수 값 변경시 변수에 들어있던 기존 값은 삭제
