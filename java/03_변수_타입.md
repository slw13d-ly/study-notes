## 변수 타입(type)

[소스](src/variable/Var7.java)

- 데이터를 다루는 종류에 따른 다양한 형, 형식
- 지정된 타입에 맞는 값 사용 - 다르면 컴파일 에러

  | 변수 타입 | 설명                    | 예 |
  |---|-----------------------|---|
  | `int` | 정수                    | 1, 100, 1000 |
  | `double` | 실수                    | 0.2, 1.5, 100.121, -10 |
  | `boolean` | 참 / 거짓 판단             | `true`, `false` 사용 |
  | `char` | 문자 하나. 작은따옴표 `' '` 사용 | 'A', '가' |
  | `String` | 문자열. 큰따옴표 `" "` 사용    | "홍길동", "Hello java" |
  
  - `String` - 클래스 공부 때 더 자세히

### 리터럴(literal)

- 코드에서 개발자가 직접 적은 고정값
  - `100`, `10.5`, `true`, `'A'`, `"Hello java"`

  ```java
    int a = 100; // 정수 리터럴
    double b = 10.5; // 실수 리터럴
    boolean c = true; // 불리언 리터럴
    char d = 'A'; // 문자 하나 리터럴
    String e = "Hello java"; // 문자열 리터럴
  ```
  
### 변수 타입 

[소스](src/variable/Var8.java)

#### 정수 타입
| 변수 타입 | 메모리 크기           | 설명                      |
|------|------------------|-------------------------|
| byte | 1byte, $2^{8}$   | 파일 전송시 사용               |
| short | 2byte, $2^{16}$  |                         |
| char | 2byte, $2^{16}$  | 유니코드                    |
| `int` | 4byte, $2^{32}$  |                         |
| `long` | 8byte, $2^{64}$  | 리터럴 뒤 'L'로 long 타입임을 명시 |

- `byte`, `short`, `int`, `long` - 부호(signed) 있는 정수 타입. 최상위 bit는 부호 bit

#### 실수 타입
| 변수 타입 | 메모리 크기          | 설명                               |
|---------|-----------------|----------------------------------|
| float   | 4byte, $2^{32}$ | 부호(1bit) + 지수(8bit) + 기수(23bit)  |
| `double` | 8byte, $2^{64}$ | 부호(1bit) + 지수(11bit) + 기수(52bit) |

- `float` - 리터럴 뒤 'f/F'로 float 타입임을 명시

#### 논리 타입
| 변수 타입 | 메모리 크기 | 설명                            |
|----------|--------|-------------------------------|
| `boolean` | 1byte, $2^{8}$ | 참 / 거짓 판단. `true`, `false` 사용 |

#### 문자열 타입
| 변수 타입 | 메모리 크기          | 설명          |
|----------|-----------------|-------------|
| `char`    | 2byte, $2^{16}$ | 문자 하나, 유니코드 |
