## 변수 타입(type)

[소스](src/variable/Var7.java)

- 데이터를 다루는 종류에 따른 다양한 형식
- 지정된 타입에 맞는 값 사용 - 다르면 컴파일 에러

  | 변수 타입 | 설명                    | 예 |
  |---|-----------------------|---|
  | `int` | 정수                    | 1, 100, 1000 |
  | `double` | 실수                    | 0.2, 1.5, 100.121, -10 |
  | `boolean` | 참 / 거짓 판단             | `true`, `false` 사용 |
  | `char` | 문자 하나. 작은따옴표 `' '` 사용 | 'A', '가' |
  | `String` | 문자열. 큰따옴표 `" "` 사용    | "홍길동", "Hello java" |
  
  - `String` - 클래스 공부 때 더 자세히

### 리터럴(literal)

- 코드에서 개발자가 직접 적은 고정값
  - `100`, `10.5`, `true`, `'A'`, `"Hello java"`

  ```java
    int a = 100; // 정수 리터럴
    double b = 10.5; // 실수 리터럴
    boolean c = true; // 불리언 리터럴
    char d = 'A'; // 문자 하나 리터럴
    String e = "Hello java"; // 문자열 리터럴
  ```
