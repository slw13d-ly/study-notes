### 변수 선언

- 변수 - 하나의 값을 저장할 수 있는 메모리 번지에 붙여진 이름
  - 변수를 통해 프로그램이 메모리 번지에 값을 저장 + 읽기 가능

- 변수 선언 방법
  - 첫 글자 문자. 중간부터 문자, 숫자, $, \_ 등 가능
  - 첫 문자는 소문자. 캐멀 스타일로 작성
    - 캐멀(camel) 스타일 - 낙타 등, 대소문자를 섞는 것
      - 클래스명 - Week.java, MamberGrade.java, ProductKind.java
      - 변수명 - score, mathScore, sportsCar

- 변수 선언
  - 저장되는 값의 타입과 이름만 결정, 메모리의 할당된 것 x
  - 최초의 값이 대입될 시 메모리에 할당, 해당 메모리에 값 저장

- 변수 값 저장
  - 대입 연산자( = ) 사용 - 우측 값을 좌측 변수에 대입
  - 변수 초기화 - 변수에 최초로 값을 대입하는 행위
    - 초기화되지 않은 변수는 메모리에 할당되지 않음 - 컴파일 에러
  - 초기값 - 변수에 최초로 대입한 값
    - 변수 선언과 동시에 대입 가능

      ```java
      int score = 90;
      ```

    - 잘못된 경우

      ```java
      int value;

      int result = value + 10
      ```
