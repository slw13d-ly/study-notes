## 연산자(operator)

```text
3 + 4
a + b
```

- 연산자(operator) - 연산 기호 `+`, `-`
- 피연산자(operand) - 연산 대상 `3`, `4`, `a`, `b`

### 산술 연산자

[소스](src/operator/Operator1.java)

| 연산식 | 설명          |
|-----|-------------|
| +   | 덧셈 연산       |
| -   | 뺄셈 연산       |
| *   | 곱셈 연산       |
| /   | 나눗셈 연산      |
| %   | 나눗셈의 나머지 연산 |

- 숫자는 0으로 나눌 수 없음(수학적 허용 x)
  - 0 대입시 컴파일 에러

### 문자열 더하기

[소스](src/operator/Operator2.java)

- 문자열에 `+` 연산자를 사용해 문자 연결 가능

  ```text
  String result1 = "hello " + "world";
  ```
  - "hello " 문자열 + "world" 문자열 결합 → `"hello world"`

  ```text
  String result2 = s1 + s2;
  ```
  - s1과 s2 변수의 문자열을 읽음
  - "string1" 문자열 + "string2" 문자열 결합 → `"string1string2"`

#### 문자열 + 숫자

- 문자와 숫자를 더하면 `숫자를 문자열로 변경 후` 서로 더함

  ```text
  "a + b = " + 10
  
  //계산 과정
  "a + b = "(String) + 10(int)
  "a + b = "(String) + 10(int → String) //숫자를 문자열로 변경
  "a + b = " + "10" //문자열 + 문자열
  
  "a + b = 10" //결과
  ```

  ```text
  str(String) + num(int)
  ```
  - 변수에 담겨 있어도 숫자 → 문자 변경

### 증감 연산자

[소스](src/operator/OperatorAdd1.java)

- `++`, `--` - 변수의 값을 1만큼 증가 및 감소 
- `a = a + 1` = `++a`

#### 전위/후위 증감 연산자

[소스](src/operator/OperatorAdd2.java)

- 증감 연산자를 피연산자 앞에 두거나 뒤에 두어, 연산자의 위치에 따라 연산이 수행되는 시점이 달라짐
  - `++a` - 전위(Prefix) 증감 연산자. a의 증감 연산이 먼저 진행
  - `a++` - 후위(Postfix) 증감 연산자. a에 대한 다른 연산이 먼저 수행 후 증감 연산 수행
    - 증감 연산자의 단독 사용의 경우 다른 값이 없어 본인 값만 증가. 전위, 후위 결과 같음

### 비교 연산자

[소스](src/operator/Comp1.java)

- 두 값을 비교하는 연산자. 주로 뒤에서 설명하는 조건문과 함께 사용

  | 연산식  | 설명           |

  | `<`  | 작다           |
  | `>=` | 크거나 같다       |
  | `<=` | 작거나 같다       |
  - `=` - 대입 연산자 / `==` - 동등성 확인 비교 연산자

### 논리 연산자

- `boolean`형인 `true`, `false` 비교 시 사용

  | 연산식        | 설명                                      |

  | `\|\|`(OR) | 두 피연산자 중 하나라도 참이면 참 반환, 둘 다 거짓이면 거짓 반환  |
  | `!`        | 피연산자의 논리적 부정, 참이면 거짓, 거짓이면 참 반환         |

### 대입 연산자

- `=` - 값을 변수에 할당하는 연산자
- 예) `int a = 1` - `a`라는 변수에 `1`값 할당
