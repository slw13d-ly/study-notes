## 연산자(operator)

```text
3 + 4
a + b
```

- 연산자(operator) - 연산 기호 `+`, `-`
- 피연산자(operand) - 연산 대상 `3`, `4`, `a`, `b`

### 산술 연산자

[소스](src/operator/Operator1.java)

| 연산식 | 설명          |
|-----|-------------|
| +   | 덧셈 연산       |
| -   | 뺄셈 연산       |
| *   | 곱셈 연산       |
| /   | 나눗셈 연산      |
| %   | 나눗셈의 나머지 연산 |

- 숫자는 0으로 나눌 수 없음(수학적 허용 x)
  - 0 대입시 컴파일 에러
